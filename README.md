# SGDI - Sistema de Gestão de Demandas Internas

Sistema para gerenciar demandas internas da empresa.

## Como rodar

```bash
python init_db.py
python app.py
```

Acesse: http://localhost:5000

## Funcionalidades

- Criar demandas
- Editar demandas
- Deletar demandas
- Visualizar detalhes
- Comentários
- Prioridades com 3 níveis padrão (Alta, Média e Baixa)
- Cadastro, edição e exclusão de prioridades com cor e ordem
- Ordenação de demandas por prioridade e data de criação
- Campos obrigatórios: Título e Prioridade
